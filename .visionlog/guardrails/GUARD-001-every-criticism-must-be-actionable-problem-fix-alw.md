title: "Every criticism must be actionable \u2014 problem + fix, always"
status: "active"
date: "2026-03-24"
---

"This sucks" is not a finding. "This sucks because X, fix by doing Y" is. If you can't articulate the fix, you haven't understood the problem well enough to report it.
